BIOS/UEFI

What is BIOS/UEFI?

BIOS (Basic Input/Output System) and its modern replacement UEFI (Unified Extensible Firmware Interface) are low-level software that starts when you power on your computer, before the operating system loads. They initialize and test hardware components, then load the bootloader to start the OS.

Key Functions

  • POST (Power-On Self-Test): Checks hardware components before boot
  • Hardware Initialization: Sets up and configures hardware
  • Boot Device Selection: Determines which device to boot from
  • CMOS Setup: Provides interface for hardware configuration
  • Hardware Monitoring: Tracks temperatures, voltages, and fan speeds

BIOS vs UEFI

Feature BIOS UEFI
Interface Text-based Graphical, mouse support
Boot Time Slower Faster
Disk Support MBR only (up to 2TB) GPT (disks over 2TB)
Security Basic Secure Boot feature

Troubleshooting Tips

  • Can't enter BIOS: Try different keys (Del, F2, F10, F12) or disable Fast Boot
  • Boot device not recognized: Check boot order and SATA mode (AHCI/IDE)
  • CMOS settings reset: Replace CMOS battery if settings don't persist
  • Secure Boot issues: Disable Secure Boot or update BIOS for compatibility
