BIOS/UEFI

The firmware interface between hardware and operating system

Description and Function

BIOS (Basic Input/Output System) and UEFI (Unified Extensible Firmware Interface) are firmware interfaces that initialize hardware during boot and provide runtime services.

Key functions:

  • POST: Power-On Self-Test checks hardware
  • Hardware Initialization: Prepares hardware for OS
  • Bootloader: Loads the operating system
  • CMOS Setup: Configuration interface

UEFI advantages:

  • Faster boot times
  • Support for >2TB drives
  • Secure Boot security feature
  • Graphical interface

Troubleshooting Tips

  • Can't enter BIOS: Try different keys (Del, F2, F10)
  • Boot issues: Check boot order settings
  • Settings reset: Replace CMOS battery
