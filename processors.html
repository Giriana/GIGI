Central Processing Unit (CPU)

The brain of the computer that performs calculations

Description and Function

The CPU executes instructions from computer programs by performing basic arithmetic, logic, and I/O operations.

Key components:

  • ALU: Arithmetic Logic Unit
  • Control Unit: Directs operations
  • Cache: Fast memory for frequent data

Performance factors:

  • Cores: Physical processors (4-32 in modern CPUs)
  • Threads: Virtual cores via hyperthreading
  • Clock Speed: GHz (billions of cycles per second)
  • Cache: L1/L2/L3 cache sizes

Troubleshooting Tips

  • Overheating: Check cooler installation
  • No boot: Verify power connections
  • Performance issues: Check for thermal throttling
