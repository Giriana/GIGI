Random Access Memory (RAM)

Temporary storage for active programs and data

Description and Function

RAM is volatile memory that temporarily stores data the CPU needs quick access to while performing tasks.

Key characteristics:

  • Volatile: Loses data when powered off
  • Fast: Much faster than SSDs/HDDs
  • Capacity: Typically 8GB-32GB in modern systems

Common types:

  • DDR4: Current standard (2014-present)
  • DDR5: Newer standard with higher speeds
  • SO-DIMM: Laptop memory form factor

Troubleshooting Tips

  • No boot: Reseat RAM modules
  • Crashes: Run memory diagnostics
  • Not recognized: Check compatibility
