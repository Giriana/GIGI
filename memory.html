Memory (RAM)

What is RAM?

Random Access Memory (RAM) is a form of computer memory that can be read and changed in any order, typically used to store working data and machine code. It is volatile memory, meaning it loses its contents when power is removed.

Types of RAM

  • DRAM (Dynamic RAM): Needs constant refreshing, used for main system memory
  • SRAM (Static RAM): Faster but more expensive, used for cache
  • SDRAM (Synchronous DRAM): Synchronized with system clock
  • DDR SDRAM (Double Data Rate): Current standard (DDR4, DDR5)

RAM Specifications

  • Capacity: Measured in GB (4GB, 8GB, 16GB, etc.)
  • Speed: Measured in MHz (2400MHz, 3200MHz, etc.)
  • Latency: CAS latency (CL) timings (CL16, CL18, etc.)
  • Form Factor: DIMM (desktop), SODIMM (laptop)

Troubleshooting Tips

  • No boot/beeps: Reseat RAM modules and check compatibility
  • BSODs/crashes: Run memory diagnostics (Windows Memory Diagnostic)
  • Not all RAM detected: Check for OS limitations (32-bit vs 64-bit)
  • Performance issues: Ensure RAM is running at correct speed in BIOS
