Motherboards

The main printed circuit board (PCB) in general-purpose computers

Description and Function

A motherboard is the main printed circuit board (PCB) that holds and allows communication between crucial electronic components like the CPU and memory.

Key components:

  • CPU Socket: Holds the processor
  • RAM Slots: For memory modules
  • Chipset: Manages data flow
  • PCIe Slots: For expansion cards
  • SATA/M.2: For storage devices

Form factors:

  • ATX: Standard size (12 × 9.6 in)
  • Micro-ATX: Smaller (9.6 × 9.6 in)
  • Mini-ITX: Compact (6.7 × 6.7 in)

Troubleshooting Tips

  • No power: Check 24-pin and CPU power connectors
  • No display: Reseat RAM and GPU
  • USB ports not working: Update drivers
