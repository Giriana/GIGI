Video & Expansion Cards

Components that add functionality to your computer

Description and Function

Expansion cards add capabilities to computers through PCIe slots on the motherboard.

Common types:

  • Graphics Cards (GPU): Handle visual processing
  • Sound Cards: Enhance audio capabilities
  • Network Cards: Add wired/wireless networking
  • Capture Cards: Record video input

GPU components:

  • GPU Chip: The main processor
  • VRAM: Video memory (GDDR5/GDDR6)
  • Cooling System: Fans or liquid cooling
  • Power Connectors: 6-pin/8-pin PCIe

Troubleshooting Tips

  • No display: Check power connections
  • Artifacts: May indicate failing GPU
  • Driver issues: Update graphics drivers
