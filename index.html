Computer Cases & Form Factors

Description and Function

The computer case is the enclosure that contains most of the components of a computer. It serves several important functions:

  • Protects internal components from physical damage and dust
  • Provides structural support for mounting components
  • Helps with cooling by allowing proper airflow
  • Shields against electromagnetic interference

Form Factors

Computer cases come in different sizes and shapes known as form factors. The most common include:

  • ATX: The standard full-tower size for most desktop PCs
  • Micro-ATX: A smaller version of ATX with fewer expansion slots
  • Mini-ITX: Very compact form factor for small PCs
  • Tower: Vertical orientation (full, mid, mini sizes)
  • Desktop: Horizontal orientation

Troubleshooting Tips

  • Overheating: Ensure proper airflow and clean dust filters regularly
  • Noise: Check for loose components or failing case fans
  • Component doesn't fit: Verify case supports your motherboard and component form factors
  • Front panel not working: Check connections to motherboard

Motherboards

Description and Function

The motherboard is the main printed circuit board (PCB) in a computer. It holds and allows communication between many of the crucial electronic components of a system, such as the CPU and memory, and provides connectors for other peripherals.

Key Components on a Motherboard

  • CPU Socket: Where the processor is installed
  • RAM Slots: For memory modules
  • Chipset: Manages data flow between components
  • Expansion Slots: PCIe slots for graphics cards and other add-ons
  • Storage Connectors: SATA and M.2 for drives
  • Power Connectors: To receive power from PSU
  • I/O Ports: USB, audio, network, etc.

Troubleshooting Tips

  • No power: Check power connections and front panel headers
  • No display: Verify CPU and RAM are properly seated
  • USB ports not working: Check for bent pins or driver issues
  • Random crashes: Look for swollen capacitors or overheating

BIOS/UEFI

Description and Function

BIOS (Basic Input/Output System) and its modern replacement UEFI (Unified Extensible Firmware Interface) are low-level software that starts when you power on your computer before the operating system loads. They perform hardware initialization and provide runtime services for operating systems.

Key Functions

  • POST (Power-On Self-Test): Checks hardware components
  • Hardware Initialization: Sets up and configures hardware
  • Bootloader: Locates and loads the operating system
  • CMOS Setup: Provides configuration interface
  • Hardware Abstraction: Provides consistent way for OS to interact with hardware

Troubleshooting Tips

  • No POST beep: Check power supply and motherboard connections
  • CMOS reset: Replace CMOS battery if settings aren't saved
  • Boot issues: Verify boot order and boot mode (UEFI/Legacy)
  • Update failures: Never interrupt a BIOS/UEFI update

Memory (RAM)

Description and Function

Random Access Memory (RAM) is a form of computer memory that can be read and changed in any order, typically used to store working data and machine code. It is volatile memory, meaning it requires power to maintain the stored information.

Types of RAM

  • DRAM (Dynamic RAM): Needs constant refreshing, used for main memory
  • SRAM (Static RAM): Faster but more expensive, used for cache
  • SDRAM (Synchronous DRAM): Synchronized with system clock
  • DDR SDRAM: Double Data Rate versions (DDR2, DDR3, DDR4, DDR5)

Troubleshooting Tips

  • System won't boot: Reseat RAM modules or try one stick at a time
  • Blue screens/random crashes: Run memory diagnostics
  • Not all RAM recognized: Check for limitations in OS or motherboard
  • Performance issues: Ensure RAM is running at correct speed in BIOS

Processors (CPUs)

Description and Function

The Central Processing Unit (CPU) is the electronic circuitry that executes instructions comprising a computer program. It performs the basic arithmetic, logic, controlling, and input/output (I/O) operations specified by the instructions.

Key Characteristics

  • Cores: Modern CPUs have multiple processing cores
  • Clock Speed: Measured in GHz, how many cycles per second
  • Cache: Small, fast memory close to the CPU
  • Threads: Ability to handle multiple instruction streams
  • Thermal Design Power (TDP): Heat dissipation requirements

Troubleshooting Tips

  • Overheating: Check cooler installation and thermal paste
  • No POST: Verify CPU is compatible with motherboard
  • System instability: Check for adequate power supply
  • Performance issues: Monitor for thermal throttling

Video & Expansion Cards

Description and Function

Expansion cards are printed circuit boards that can be inserted into an expansion slot of a computer motherboard to add functionality to a computer system. The most common type is the graphics card (GPU), but there are many others.

Types of Expansion Cards

  • Graphics Cards (GPU): Handles rendering of images and video
  • Sound Cards: Provides audio input and output capabilities
  • Network Cards: For wired or wireless network connectivity
  • Storage Controllers: Adds additional storage interfaces
  • Capture Cards: For recording video input

Troubleshooting Tips

  • No display: Ensure card is properly seated and powered
  • Driver issues: Install latest drivers from manufacturer
  • Artifacts on screen: Could indicate failing GPU
  • Card not recognized: Check BIOS settings for PCIe slots

Cooling Systems

Description and Function

Computer cooling is required to remove the waste heat produced by computer components to keep them within permissible operating temperature limits. Components that are susceptible to temporary malfunction or permanent failure if overheated include CPUs, GPUs, and power supplies.

Types of Cooling

  • Air Cooling: Uses heatsinks and fans
  • Liquid Cooling: Uses coolant circulated through a loop
  • Passive Cooling: Uses only heatsinks without fans
  • Phase-change Cooling: Advanced cooling like refrigeration
  • Case Fans: For overall system airflow

Troubleshooting Tips

  • Overheating: Check fan operation and dust buildup
  • Noisy fans: May need cleaning or replacement
  • Liquid cooling leaks: Immediately power down system
  • Thermal throttling: Reapply thermal paste if needed

Power Supply (PSU)

Description and Function

A power supply unit (PSU) converts mains AC to low-voltage regulated DC power for the internal components of a computer. Modern personal computers universally use switched-mode power supplies.

Key Specifications

  • Wattage: Total power output capacity
  • Efficiency Rating: 80 Plus certification levels (White, Bronze, Gold, etc.)
  • Modularity: Non-modular, semi-modular, or fully modular cables
  • Connectors: Types and number of power connectors
  • Form Factor: ATX, SFX, TFX, etc.

Troubleshooting Tips

  • No power: Check power cable and switch, test outlet
  • Random shutdowns: May indicate insufficient wattage
  • Burning smell: Immediately disconnect power
  • Coil whine: Usually harmless but can indicate issues
