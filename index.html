Main Computer Components

Motherboard

The main circuit board that connects all components

Processor (CPU)

The brain of your computer that performs calculations

Memory (RAM)

Temporary storage for active programs and data

Graphics Card (GPU)

Handles visual output and accelerates graphics processing

Power Supply (PSU)

Provides power to all components in your system

Cooling System

Keeps components at safe operating temperatures

Cases & Form Factors

Enclosure that houses and protects all components

BIOS/UEFI

Firmware that initializes hardware during boot
