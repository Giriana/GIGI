Computer Components Overview

A computer is made up of several key components that work together to perform tasks. Below you'll find detailed information about each major component.

Motherboard

The motherboard is the main printed circuit board (PCB) in a computer. It holds and allows communication between many of the crucial electronic components of a system, such as the central processing unit (CPU) and memory, and provides connectors for other peripherals.

Key Specifications:

  • Form Factor: ATX, microATX, mini-ITX
  • Chipset: Determines CPU compatibility and features
  • Expansion Slots: PCIe x16, PCIe x1, M.2
  • RAM Slots: Typically 2-8 DIMM slots
Learn More About Motherboards

Processor (CPU)

The central processing unit (CPU) is the electronic circuitry that executes instructions comprising a computer program. The CPU performs basic arithmetic, logic, controlling, and input/output (I/O) operations specified by the instructions in the program.

Key Specifications:

  • Cores/Threads: Modern CPUs have 4-32 cores
  • Clock Speed: Measured in GHz (e.g., 3.6GHz)
  • Cache: L1, L2, L3 cache memory
  • TDP: Thermal Design Power (35W-125W)
